## Project Overview

IZIWEB is a corporate website built with a custom WordPress theme.  
The project focuses on clean architecture, scalability, and performance.

The website includes dynamic content management using Advanced Custom Fields (ACF) and custom post types.

- jQuery
- Swiper.js
- MySQL
